# Idman Monitor v3 — safe mode

Эта версия сделана после ошибки Render: `Ran out of memory (used over 512MB)`.

Что изменилось:
- Уменьшена нагрузка на память.
- Главная + максимум 2 релевантных раздела.
- Максимум 12 кандидатных новостей с источника за запуск.
- Финальный доскан временно отключён для стабильности.
- Добавлена очистка памяти после каждого источника.
- Добавлена защита от слишком тяжёлых страниц.
- Таймаут сайта снижен до 5 секунд.

После суток теста можно будет аккуратно увеличить глубину.
