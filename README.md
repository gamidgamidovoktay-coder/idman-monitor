# Idman Monitor v2

Более быстрая и понятная версия.

Что изменилось:
- Добавлены подробные логи по каждому источнику.
- Уменьшены таймауты, чтобы запуск не висел слишком долго.
- Уменьшена глубина проверки ссылок на первом этапе.
- Убрано предупреждение XMLParsedAsHTMLWarning.
- Финальный доскан временно сокращён до 60 секунд для тестов.

После стабильной работы можно будет снова увеличить глубину.
