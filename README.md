# Idman Monitor v4

Полная редакционная версия, собранная на базе рабочей v3.

## Что включено

- Один проход без Final scan.
- Окно свежести: последние 60 минут.
- Память отправленных новостей: повторно не отправляет уже отправленное.
- Дедупликация одной и той же новости внутри письма.
- Фильтрация мусорных страниц: категории, разделы, служебные страницы, APA weather/army/media, Trend business/economy и др.
- Приоритеты:
  - 🔥 Важно: трансферы, травмы, сборная, еврокубки, решения/скандалы, интервью.
  - 🟡 Среднее: результаты, туры, турниры.
  - ⚪ Низкое: остальное.
- Разделы письма:
  - ⚽ Футбол
  - 🏅 Другие виды спорта
- Максимум 50 новостей в письме.
- Мелкие сайты оставлены.
- Логи по каждому источнику сохранены.
- Защита от тяжелых страниц и ограничение нагрузки сохранены.

## Как обновить

1. Загрузить файлы поверх старых в GitHub.
2. Commit changes.
3. Дождаться build в Render.
4. Trigger Run.
